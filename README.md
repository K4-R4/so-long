# so-long
### Description
42の課題で制作した2Dの脱出ゲームです。指定のフォーマットのマップファイルを読み取り描画します。
```bash: medium_map.ber
1111111111111
10010000000C1
1000011111001
1P0011E000001
1111111111111
```
- 1: Wall
- 0: Empty
- E: Exit
- C: Collectible
- P: Player
### Usage
```bash
git clone https://github.com/K4-R4/so-long
cd so-long
make
```
